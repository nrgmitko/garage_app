# Car Management API

## Описание

Car Management API е backend RESTful услуга за система за управление на автопаркове.
API-то позволява управление на автомобили, сервизи и заявки за поддръжка.

Системата поддържа:

* управление на автомобили
* управление на сервизи
* създаване и управление на заявки за поддръжка
* проверка за капацитет на сервизите
* справка за заявки по месеци

Frontend приложение използва това API без нужда от модификации.

# Инсталация

## 1. Клониране на проекта

```
git clone <repository-url>

```

---

```

```

# Стартиране на базата данни

Проектът използва **MySQL чрез Docker**.

Примерен docker контейнер:

```
docker run --name car-mysql \
-e MYSQL_ROOT_PASSWORD=1234qwer \
-e MYSQL_DATABASE=car_management_db \
-p 3306:3306 \
-d mysql:8
```

# Основни функционалности

## Управление на автомобили

Позволява:

* създаване на автомобил
* обновяване на автомобил
* изтриване на автомобил
* извличане на автомобил по ID
* списък с автомобили с филтри

Филтри:

* марка
* година
* сервиз

---

## Управление на заявки за поддръжка

Позволява:

* създаване на заявка
* обновяване на заявка
* изтриване на заявка
* извличане по ID
* списък със заявки

### Проверка на капацитет

При създаване или редактиране на заявка системата проверява дали сервизът има свободни места за избраната дата.

Ако няма свободни места се връща грешка.

---

## Справка за заявки по месеци

Endpoint:

```
GET /maintenance/monthlyRequestsReport
```

Параметри:

* `garage_id`
* `start_month`
* `end_month`

Пример:

```
/maintenance/monthlyRequestsReport?garage_id=1&start_month=2024-01&end_month=2024-12
```

Резултатът връща:

* година
* месец
* брой заявки

Месеците без заявки се връщат с **0**.
